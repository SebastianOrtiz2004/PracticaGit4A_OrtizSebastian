Bases de datos conocidas

- MySQL: Sistema de gestión de bases de datos relacional popular por su simplicidad y eficiencia. -PostgreSQL: Base de datos relacional avanzada con soporte para extensibilidad y estándares SQL estrictos. - MongoDB: Base de datos NoSQL orientada a documentos, ideal para manejar grandes volúmenes de datos no estructurados. - Oracle Database: Sistema de gestión de bases de datos relacional robusto, diseñado para grandes empresas y alta disponibilidad.
